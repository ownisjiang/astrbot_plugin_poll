# 📊 PollStar - AstrBot 投票插件

在聊天中创建投票、参与投票、查看结果。

## ✨ 功能

- ✅ **创建投票** — 支持单选/多选，随时发起
- ✅ **匿名投票** — `--anon` 参数，投票人可见
- ✅ **定时截止** — `--time N` 参数，N 分钟后自动截止
- ✅ **实时结果** — 投票后自动显示最新结果，带进度条
- ✅ **持久化** — 数据存本地文件，重启不丢
- ✅ **零依赖** — 无需额外安装 Python 包

## 📋 命令

| 命令 | 说明 |
|------|------|
| `/poll help` | 查看帮助 |

### 创建投票
```
/poll create "问题" "选项1" "选项2" ...

参数:
  --multi    多选 (默认单选)
  --anon     匿名投票
  --time N   N 分钟后截止

示例:
  /poll create "今晚吃什么?" "火锅" "烤肉" "炒菜"
  /poll create "选组长" "小明" "小红" --anon
  /poll create "周末去哪" "海边" "爬山" "露营" --time 60
```

### 参与 / 查看
```
/poll vote <ID> <编号>      参与投票
/poll result <ID>          查看结果
/poll list                  所有活跃投票
/poll close <ID>           关闭投票（仅创建者）
/vote <ID> <编号>           快捷投票
```
